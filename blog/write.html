How to publish

  1. Write your post using the rich text editor above.
  2. Click Export HTML to download the complete blog post file.
  3. Place the downloaded .html file in the blog/posts/ directory.
  4. Add the post entry to blog/posts/posts.json (the export will show you the JSON to copy).
  5. Commit and push to GitHub. Your post will appear on the blog automatically.
